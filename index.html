ForeverAlone


Social website for meetings, made with PHP, javascript, html, css, Photoshop for creating the image layout, WAMP and PostgreSQL.


Users can:

  • Create profile
  • Send wedding proposal
  • See matching users and their description
  • Invite matching users for meetings
  • Accept or decline meetings
  • See invitations,confirmed meetings and budgets for weddings
  • Change preferences for a partner
  • Change profile data

Admins can:

  • Create users and remove
  • Search users
  • Change user info
  • Create parties
  • Invite users for parties
  • Generate reports about meetings, parties and productivity
  • See pending requested weddings and send budget

Angular restaurant (In Progress)


Website for a restaurant, made with Angular, NgRx Store, CSS, Bootstrap.


Users can:

  • Search for food items
  • Drag and drop food items in the cart
  • Undo / Redo dropping
  • Complete order
  • See the items ready to be fetched

Bartenders can:

  • Update inventory
  • See which tables ordered and what was ordered
  • Update orders to be fetched (Deliver)
  • Cancel orders
  • Clean tables (orders)
